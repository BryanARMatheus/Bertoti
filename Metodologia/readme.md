## Meus Projetos

### Em 2025-1
O projeto que desenvolvemos no primeiro semestre de 2025 foi um sistema de registro de movimentações para a empresa Altave, o projeto foi feito utilizando vue.js juntamente com a biblioteca ant design vue, aplicamos java e springboot no backend do projeto.

A empresa parceira Altave foca no desenvolvimento de balões para monitoramento inteligente de areas extensas, alem de tecnologias utilizando Inteligencia Artificial e video analytics para identificação de riscos no local de trabalho os catastrofes ambientais.

Então nos foi apresentado o seguinte problema: nos locais de manutenção de navios se apresentou a necessidade de monitorar os horarios de entrada e saida do local de trabalho dos funcionarios de empresas de terceiras, portanto o produto deveria auxiliar no controle e administração dos horarios dos funcionarios.

Nossa solução foi uma aplicação web que permitia o cadastro e pesquisa de funcionarios, empresas, contudo sua função principal era o registro e exibição de pontos de entrada e saida de cada funcionario, alem disso o produto providenciava gráficos e outras informações importantes por ultimo tambem era possivel extração de registros em formato de planilha. 

[Repositorio do projeto](https://github.com/SQLutions-FATEC/API-3-Semestre)

#### Tecnologias Utilizadas
- **Vue.js:** Framework de JavaScript escolhido para o projeto, utilizado para construir toda a camada visual para o projeto. 
- **Java:** Linguagem principal utilizada no desenvolvimento do backend do projeto, uma linguagem robusca focada em objetos com incrivel capacidade multiplataforma. 
- **Springboot:** Um framework de Java que auxilia e facilita o desenvolvimento de APIs REST, fornece varias funções para agilizar o desenvolvimento do backend do produto.
- **Docker:** Uma plataforma que foca na conteinerização de projetos, permite através do empacotamento de aplicações e dependencias garantir consistencia ao tentar executar o projeto em diferentes máquinas.
- **PostgreSQL:** O sistema de banco de dados selecionado para o projeto, é um banco de dados relacional com acesso a diversos recursos e com certo foco para integridade de dados.

#### Contribuições Pessoais
Neste projeto atuei em grande parte no Frontend do projeto, estas foram minhas principais contribuições:
- **Tela de cadastro de funcionario:** Desenvolvi as telas de cadastro de funcionario utilizando vue.js, utilizei conceitos basicos de JavaScript, aprendi sobre o framework Vue.js, sobre o Axios e como conectar com o backend em springboot a partir dele;
- **Tabela de registro de movimentações:** Ajudei no desenvolvimento na tabela de registros de movimentações, adicionei colunas, aprendi sobre formatação de dados recebidos do backend, tambem adicionei a função de editar esses registros reforçando as habilidades de utilizar os serviços do axios e enviar os dados para o backend;
- **Função de exportação de dados em .csv:** Por ultimo fiz a função exportação e formatação de todos os dados de movimentações em um formato .csv, isto me ensinou sobre as vastas funcionalidades que as bibliotecas do JavaScript conseguem realizar, sobre métodos de buffering, como a paginação das informações funcionava, como modificar um request para o backend e como formatar os dados para um arquivo .csv a partir da biblioteca utilizada.
